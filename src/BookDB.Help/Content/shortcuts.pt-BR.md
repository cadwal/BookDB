# Atalhos de teclado

Referência completa dos atalhos de teclado no BookDB.

## Janela principal

| Atalho | Ação |
|--------|------|
| Ctrl+N | Novo livro — abre a caixa de diálogo Adicionar livro |
| F2 | Editar livro — abre o livro selecionado para edição inline |
| Ctrl+D | Duplicar livro — cria uma cópia do livro selecionado |
| Ctrl+Shift+F | Pesquisa avançada — abre a caixa de diálogo de pesquisa avançada |
| Ctrl+Return | Abrir detalhes completos — abre o livro selecionado em uma janela separada |
| Ctrl+I | Catalogar por ISBN — abre o assistente de pesquisa por ISBN |
| Alt+F4 | Sair do aplicativo |

## Lista de livros

| Atalho | Ação |
|--------|------|
| Delete | Excluir livro(s) selecionado(s) — somente quando a lista de livros tem foco |
| Ctrl+A | Selecionar todos os livros na lista |
| Espaço | Alternar caixa de seleção para a linha em foco |

## Notas

- **Delete** só funciona quando a lista de livros (DataGrid) tem o foco do teclado. Se o formulário de edição ou outro painel tiver foco, Delete não excluirá livros.
- Os atalhos de teclado se aplicam apenas à janela principal. As caixas de diálogo individuais (Assistente de importação, Adicionar livro, etc.) seguem a navegação padrão por teclado (Tab para mover entre campos, Enter para confirmar, Escape para cancelar).
