# Tastenkürzel

Eine vollständige Referenz der Tastenkürzel in BookDB.

## Hauptfenster

| Kürzel | Aktion |
|--------|--------|
| Strg+N | Neues Buch — öffnet den Dialog „Buch hinzufügen" |
| F2 | Buch bearbeiten — öffnet das ausgewählte Buch zur Bearbeitung |
| Strg+D | Buch duplizieren — erstellt eine Kopie des ausgewählten Buches |
| Strg+Umschalt+F | Erweiterte Suche — öffnet den Dialog für erweiterte Suche |
| Strg+Eingabe | Vollständige Details öffnen — öffnet das ausgewählte Buch in einem separaten Fenster |
| Strg+I | Per ISBN katalogisieren — öffnet den ISBN-Suchassistenten |
| Alt+F4 | Anwendung beenden |

## Bücherliste

| Kürzel | Aktion |
|--------|--------|
| Entf | Ausgewählte Bücher löschen — nur wenn die Bücherliste den Fokus hat |
| Strg+A | Alle Bücher in der Liste auswählen |
| Leertaste | Auswahlkästchen für die fokussierte Zeile umschalten |

## Hinweise

- **Entf** funktioniert nur, wenn die Bücherliste (DataGrid) den Tastaturfokus hat. Falls das Bearbeitungsformular oder ein anderes Panel den Fokus hat, werden keine Bücher gelöscht.
- Tastenkürzel gelten nur für das Hauptfenster. Einzelne Dialoge (Importassistent, Buch hinzufügen usw.) folgen der Standard-Tastaturnavigation (Tab zum Wechseln zwischen Feldern, Eingabe zum Bestätigen, Escape zum Abbrechen).
