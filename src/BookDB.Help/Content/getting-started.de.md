# Erste Schritte

Willkommen bei BookDB — einem persönlichen Katalog für Ihre Büchersammlung. Diese Anleitung führt Sie durch die ersten Schritte.

## Ihr erstes Buch hinzufügen

Am schnellsten fügen Sie ein Buch über seine ISBN hinzu:

1. Drücken Sie **Strg+N**, oder verwenden Sie **Datei → Neues Buch**, oder die Hinzufügen-Schaltfläche in der Symbolleiste.
2. Geben oder scannen Sie die **ISBN** ein und drücken Sie **Enter**.
3. BookDB schlägt das Buch in mehreren öffentlichen Quellen nach und öffnet den Dialog **Zusammenführen und prüfen**, in dem Sie Titel, Autoren, Cover und weitere Angaben auswählen.
4. Wählen Sie eine **Sammlung** für das Buch und klicken Sie auf **Speichern** — oder **Speichern und Editor öffnen**, um mehr einzutragen.

Keine ISBN zur Hand? Wählen Sie im selben Dialog **Angaben manuell eingeben** und tippen Sie Titel, Autoren und weitere Felder selbst ein.

## Eine vorhandene Sammlung importieren

Sie haben bereits eine Bibliothek in Readerware? Sie müssen sie nicht neu erfassen:

- Verwenden Sie **Datei → Readerware-Sicherung importieren…**, oder die Schaltfläche **Readerware-Sicherung importieren** auf dem Bildschirm der leeren Bibliothek.
- Verweisen Sie den Assistenten auf eine Readerware **Sicherungs-.zip** oder den entpackten **Sicherungsordner**.
- Der Assistent zeigt eine Vorschau des Gefundenen, lässt Sie eine Zielsammlung wählen und importiert dann.

Eine Schritt-für-Schritt-Anleitung finden Sie im Tab **Importleitfaden**.

## Mit einer Datenbank verbinden

Standardmäßig hält BookDB Ihre Bibliothek in einer lokalen Datei auf diesem Computer — es ist nichts weiter einzurichten.

Um eine Bibliothek geräteübergreifend zu teilen oder PostgreSQL / MySQL zu verwenden, öffnen Sie **Einstellungen → Datenbank** (oder die Schaltfläche **Mit einer Datenbank verbinden** auf dem Bildschirm der leeren Bibliothek) und geben Sie die Verbindungsdaten ein. Der Tab **Entfernte Datenbanken** erläutert die Optionen.

## Ihre Bibliothek ordnen

- **Sammlungen** gruppieren Ihre Bücher (Belletristik, Sachbuch, Comics …). Legen Sie eine beim Hinzufügen fest und ändern Sie sie jederzeit durch Bearbeiten des Buchs.
- **Bearbeiten** Sie ein Buch, um Angaben, Cover und Mitwirkende anzupassen. Autoren- und andere Mitwirkendennamen greifen beim Tippen auf vorhandene Personen zurück, damit dieselbe Person nicht doppelt angelegt wird.
- **Suchen und filtern** Sie über die Symbolleiste, um alles zu finden, während Ihre Bibliothek wächst.

## Wie es weitergeht

- **Tastenkürzel** — alles schneller erledigen.
- **Feldlexikon** — was jedes Feld bedeutet.
- **Importleitfaden** — Import im Detail.
- **Über Datenquellen** — woher die Buchangaben stammen.
- **Entfernte Datenbanken** — eine Bibliothek geräteübergreifend teilen.

Diese Hilfe können Sie jederzeit mit **F1** erneut öffnen.
