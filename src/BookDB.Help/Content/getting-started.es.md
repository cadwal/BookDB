# Primeros pasos

Bienvenido a BookDB, un catálogo personal para tu colección de libros. Esta guía te acompaña en lo primero que querrás hacer.

## Añade tu primer libro

La forma más rápida de añadir un libro es mediante su ISBN:

1. Pulsa **Ctrl+N**, o usa **Archivo → Nuevo libro**, o el botón de añadir de la barra de herramientas.
2. Escribe o escanea el **ISBN** y pulsa **Intro**.
3. BookDB busca el libro en varias fuentes públicas y abre el diálogo **Combinar y revisar**, donde eliges el título, los autores, la portada y otros datos que conservar.
4. Elige una **colección** para el libro y haz clic en **Guardar** — o en **Guardar y abrir el editor** para completar más datos.

¿No tienes el ISBN? Elige **Introducir los datos manualmente** en el mismo diálogo y escribe tú mismo el título, los autores y los demás campos.

## Importa una colección existente

¿Ya tienes una biblioteca en Readerware? No hace falta volver a introducirla:

- Usa **Archivo → Importar copia de seguridad de Readerware…**, o el botón **Importar copia de seguridad de Readerware** en la pantalla de biblioteca vacía.
- Indica al asistente un **.zip de copia de seguridad** de Readerware o la **carpeta de copia de seguridad** extraída.
- El asistente muestra una vista previa de lo encontrado, te deja elegir una colección de destino y luego importa.

Consulta la pestaña **Guía de importación** para ver el proceso paso a paso.

## Conéctate a una base de datos

De forma predeterminada, BookDB guarda tu biblioteca en un archivo local de este equipo — no hay nada más que configurar.

Para compartir una biblioteca entre dispositivos, o para usar PostgreSQL / MySQL, abre **Configuración → Base de datos** (o el botón **Conectar a una base de datos** en la pantalla de biblioteca vacía) e introduce los datos de conexión. La pestaña **Bases de datos remotas** explica las opciones.

## Organiza tu biblioteca

- Las **colecciones** agrupan tus libros (Ficción, No ficción, Cómics …). Asigna una al añadir un libro y cámbiala en cualquier momento editando el libro.
- **Edita** un libro para ajustar sus datos, la portada y los colaboradores. Los nombres de autores y otros colaboradores reutilizan a las personas existentes mientras escribes, de modo que la misma persona no se duplica.
- **Busca y filtra** desde la barra de herramientas para encontrar cualquier cosa a medida que crece tu biblioteca.

## Adónde ir después

- **Atajos de teclado** — haz todo más rápido.
- **Glosario de campos** — qué significa cada campo.
- **Guía de importación** — la importación en detalle.
- **Acerca de las fuentes de datos** — de dónde proceden los datos de los libros.
- **Bases de datos remotas** — compartir una biblioteca entre dispositivos.

Puedes volver a abrir esta ayuda en cualquier momento con **F1**.
