# Atajos de teclado

Una referencia completa de los atajos de teclado en BookDB.

## Ventana principal

| Atajo | Acción |
|-------|--------|
| Ctrl+N | Nuevo libro — abre el diálogo Añadir libro |
| F2 | Editar libro — abre el libro seleccionado para edición |
| Ctrl+D | Duplicar libro — crea una copia del libro seleccionado |
| Ctrl+Mayús+F | Búsqueda avanzada — abre el diálogo de búsqueda avanzada |
| Ctrl+Intro | Abrir detalles completos — abre el libro seleccionado en una ventana separada |
| Ctrl+I | Catalogar por ISBN — abre el asistente de búsqueda por ISBN |
| Alt+F4 | Salir de la aplicación |

## Lista de libros

| Atajo | Acción |
|-------|--------|
| Supr | Eliminar libro(s) seleccionado(s) — solo cuando la lista tiene el foco |
| Ctrl+A | Seleccionar todos los libros de la lista |
| Espacio | Activar o desactivar la casilla de la fila activa |

## Notas

- **Supr** solo funciona cuando la lista de libros (DataGrid) tiene el foco del teclado. Si el formulario de edición u otro panel tiene el foco, no se eliminarán libros.
- Los atajos de teclado se aplican únicamente a la ventana principal. Los diálogos individuales (asistente de importación, añadir libro, etc.) siguen la navegación estándar de teclado (Tab para moverse entre campos, Intro para confirmar, Escape para cancelar).
