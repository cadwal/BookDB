# Raccourcis clavier

Une référence complète des raccourcis clavier dans BookDB.

## Fenêtre principale

| Raccourci | Action |
|-----------|--------|
| Ctrl+N | Nouveau livre — ouvre la boîte de dialogue Ajouter un livre |
| F2 | Modifier le livre — ouvre le livre sélectionné pour modification |
| Ctrl+D | Dupliquer le livre — crée une copie du livre sélectionné |
| Ctrl+Maj+F | Recherche avancée — ouvre la boîte de dialogue de recherche avancée |
| Ctrl+Entrée | Ouvrir les détails complets — ouvre le livre sélectionné dans une fenêtre séparée |
| Ctrl+I | Cataloguer par ISBN — ouvre l'assistant de recherche ISBN |
| Alt+F4 | Quitter l'application |

## Liste des livres

| Raccourci | Action |
|-----------|--------|
| Suppr | Supprimer le(s) livre(s) sélectionné(s) — uniquement quand la liste a le focus |
| Ctrl+A | Sélectionner tous les livres de la liste |
| Espace | Activer/désactiver la case à cocher de la ligne active |

## Remarques

- **Suppr** ne fonctionne que lorsque la liste des livres (DataGrid) a le focus clavier. Si le formulaire d'édition ou un autre panneau a le focus, aucun livre ne sera supprimé.
- Les raccourcis clavier s'appliquent uniquement à la fenêtre principale. Les boîtes de dialogue individuelles (assistant d'importation, ajout de livre, etc.) suivent la navigation clavier standard (Tab pour passer entre les champs, Entrée pour confirmer, Échap pour annuler).
