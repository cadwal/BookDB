# Scorciatoie da tastiera

Riferimento completo delle scorciatoie da tastiera in BookDB.

## Finestra principale

| Scorciatoia | Azione |
|-------------|--------|
| Ctrl+N | Nuovo libro — apre la finestra di dialogo Aggiungi libro |
| F2 | Modifica libro — apre il libro selezionato per la modifica inline |
| Ctrl+D | Duplica libro — crea una copia del libro selezionato |
| Ctrl+Shift+F | Ricerca avanzata — apre la finestra di dialogo di ricerca avanzata |
| Ctrl+Return | Apri dettagli completi — apre il libro selezionato in una finestra separata |
| Ctrl+I | Cataloga per ISBN — apre la procedura guidata di ricerca ISBN |
| Alt+F4 | Esci dall'applicazione |

## Elenco libri

| Scorciatoia | Azione |
|-------------|--------|
| Canc | Elimina libro/i selezionato/i — solo quando l'elenco libri ha il focus |
| Ctrl+A | Seleziona tutti i libri nell'elenco |
| Spazio | Attiva/disattiva la casella di selezione per la riga attiva |

## Note

- **Canc** funziona solo quando l'elenco libri (DataGrid) ha il focus da tastiera. Se il modulo di modifica o un altro pannello ha il focus, Canc non eliminerà i libri.
- Le scorciatoie da tastiera si applicano solo alla finestra principale. Le singole finestre di dialogo (Importazione guidata, Aggiungi libro, ecc.) seguono la navigazione da tastiera standard (Tab per spostarsi tra i campi, Invio per confermare, Esc per annullare).
