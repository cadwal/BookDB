# Tangentbordsgenvägar

En komplett referens för tangentbordsgenvägar i BookDB.

## Huvudfönstret

| Genväg | Åtgärd |
|--------|--------|
| Ctrl+N | Ny bok — öppnar dialogrutan Lägg till bok |
| F2 | Redigera bok — öppnar den valda boken för redigering |
| Ctrl+D | Duplicera bok — skapar en kopia av den valda boken |
| Ctrl+Skift+F | Avancerad sökning — öppnar dialogrutan för avancerad sökning |
| Ctrl+Retur | Öppna fullständiga uppgifter — öppnar den valda boken i ett separat fönster |
| Ctrl+I | Katalogisera med ISBN — öppnar guiden för ISBN-sökning |
| Alt+F4 | Avsluta programmet |

## Boklistan

| Genväg | Åtgärd |
|--------|--------|
| Delete | Ta bort valda böcker — fungerar bara när boklistan har fokus |
| Ctrl+A | Markera alla böcker i listan |
| Mellanslag | Växla kryssruta för den fokuserade raden |

## Noteringar

- **Delete** fungerar bara när boklistan (DataGrid) har tangentbordsfokus. Om redigeringsformuläret eller en annan panel har fokus tas inga böcker bort.
- Tangentbordsgenvägar gäller bara huvudfönstret. Enskilda dialogrutor (importguide, lägg till bok osv.) följer standard tangentbordsnavigering (Tab för att flytta mellan fält, Enter för att bekräfta, Escape för att avbryta).
