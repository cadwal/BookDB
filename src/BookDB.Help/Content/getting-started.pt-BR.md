# Primeiros passos

Bem-vindo ao BookDB — um catálogo pessoal para a sua coleção de livros. Este guia mostra as primeiras coisas que você vai querer fazer.

## Adicione seu primeiro livro

A forma mais rápida de adicionar um livro é pelo ISBN:

1. Pressione **Ctrl+N**, ou use **Arquivo → Novo Livro**, ou o botão de adicionar na barra de ferramentas.
2. Digite ou escaneie o **ISBN** e pressione **Enter**.
3. O BookDB busca o livro em várias fontes públicas e abre a caixa de diálogo **Mesclar e revisar**, onde você escolhe o título, os autores, a capa e outros dados a manter.
4. Escolha uma **coleção** para o livro e clique em **Salvar** — ou em **Salvar e abrir o editor** para preencher mais.

Não tem o ISBN? Escolha **Inserir os dados manualmente** na mesma caixa de diálogo e digite o título, os autores e os demais campos.

## Importe uma coleção existente

Já tem uma biblioteca no Readerware? Não precisa digitar tudo de novo:

- Use **Arquivo → Importar backup do Readerware…**, ou o botão **Importar backup do Readerware** na tela de biblioteca vazia.
- Aponte o assistente para um **.zip de backup** do Readerware ou a **pasta de backup** extraída.
- O assistente mostra uma prévia do que encontrou, deixa você escolher uma coleção de destino e então importa.

Consulte a aba **Guia de importação** para um passo a passo.

## Conecte-se a um banco de dados

Por padrão, o BookDB mantém sua biblioteca em um arquivo local neste computador — nada mais a configurar.

Para compartilhar uma biblioteca entre dispositivos, ou para usar PostgreSQL / MySQL, abra **Configurações → Banco de dados** (ou o botão **Conectar a um banco de dados** na tela de biblioteca vazia) e informe os dados de conexão. A aba **Bancos de dados remotos** explica as opções.

## Organize sua biblioteca

- As **coleções** agrupam seus livros (Ficção, Não ficção, Quadrinhos …). Defina uma ao adicionar um livro e altere quando quiser, editando o livro.
- **Edite** um livro para ajustar seus dados, a capa e os colaboradores. Os nomes de autores e outros colaboradores reutilizam pessoas existentes conforme você digita, para que a mesma pessoa não seja duplicada.
- **Pesquise e filtre** pela barra de ferramentas para encontrar qualquer coisa à medida que sua biblioteca cresce.
