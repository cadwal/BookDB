# Keyboard Shortcuts

A complete reference of keyboard shortcuts in BookDB.

## Main Window

| Shortcut | Action |
|----------|--------|
| Ctrl+N | New Book — opens the Add Book dialog |
| F2 | Edit Book — opens the selected book for inline editing |
| Ctrl+D | Duplicate Book — creates a copy of the selected book |
| Ctrl+Shift+F | Advanced Search — opens the advanced search dialog |
| Ctrl+Return | Open Full Details — opens the selected book in a separate Full Details window |
| Ctrl+I | Catalog by ISBN — opens the ISBN lookup wizard |
| Alt+F4 | Exit application |

## Book List

| Shortcut | Action |
|----------|--------|
| Delete | Delete selected book(s) — only when the book list has focus |
| Ctrl+A | Select all books in the list |
| Space | Toggle selection checkbox for the focused row |

## Notes

- **Delete** only works when the book list (DataGrid) has keyboard focus. If the edit form or another panel has focus, Delete will not delete books.
- Keyboard shortcuts apply to the main window only. Individual dialogs (Import Wizard, Add Book, etc.) follow standard keyboard navigation (Tab to move between fields, Enter to confirm, Escape to cancel).
