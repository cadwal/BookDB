# Fältlexikon

Beskrivningar av alla fält i BookDB. Fält markerade som *valfria* behöver inte fyllas i för att spara en bok.

## Titelinformation

| Fält | Beskrivning |
|------|-------------|
| Titel | Bokens primära titel. Obligatorisk. |
| Undertitel | En sekundär titelrad, visas vanligtvis under huvudtiteln på omslaget. *Valfri.* |
| Alt titel | En alternativ titel eller originaltitel (t.ex. den engelska titeln på ett översatt verk). *Valfri.* |

## Medverkande

| Fält | Beskrivning |
|------|-------------|
| Författare / Medverkande | Personerna som är involverade i att skapa boken — Författare, Redaktör, Illustratör, Designer och andra roller. Varje medverkande är en personpost kopplad till boken med en roll. |

## Publikationsuppgifter

| Fält | Beskrivning |
|------|-------------|
| Förlag | Förlagets namn. *Valfri.* |
| Utgivningsort | Utgivningsort eller utgivningsland. *Valfri.* |
| Utgivningsdatum | Utgivningsåret. Lagras som text för att stödja partiella eller ungefärliga datum som "ca. 1950". *Valfri.* |
| Copyright-datum | Upphovsrättsåret, som kan skilja sig från utgivningsdatumet i senare utgåvor. *Valfri.* |
| Format | Det fysiska formatet: Inbunden, Pocket, Storprint osv. *Valfri.* |
| Upplaga | Upplagan: Första, Andra, Reviderad osv. *Valfri.* |
| Sidor | Totalt sidantal. *Valfri.* |
| Språk | Textens språk i boken. *Valfri.* |

## Identifierare

| Fält | Beskrivning |
|------|-------------|
| ISBN | Internationellt standardboknummer (ISBN-10 eller ISBN-13). Används för metadatasökning och dubblettdetektering. *Valfri.* |
| ISSN | Internationellt standardserienummer för periodika. *Valfri.* |
| LCCN | Library of Congress kontrollnummer. *Valfri.* |
| Deweys decimalsystem | Dewey decimalkod. *Valfri.* |
| Hyllsignatur | Bibliotekssignum för hyllplacering. *Valfri.* |

## Serie

| Fält | Beskrivning |
|------|-------------|
| Serie | Den serie boken tillhör, om någon. *Valfri.* |
| Serie # | Bokens position i serien (t.ex. "3" eller "3.5"). *Valfri.* |

## Ditt exemplar

| Fält | Beskrivning |
|------|-------------|
| Exemplar | Antal fysiska exemplar du äger. Standard är 1. |
| Skick | Det fysiska skicket på ditt exemplar: Fint, Mycket bra, Bra, Godtagbart, Dåligt osv. *Valfri.* |
| Plats | Hyllan, rummet eller förvaringen där detta exemplar finns. *Valfri.* |
| Ägare | Vem som äger detta exemplar (användbart för delade samlingar). *Valfri.* |
| Signerad | Om detta är ett signerat exemplar. |
| Utgången | Om boken är markerad som utgången ur tryck. |

## Lässpårning

| Fält | Beskrivning |
|------|-------------|
| Status | Din lässtatus: Att läsa, Läser, Läst, Övergiven osv. *Valfri.* |
| Läst antal | Hur många gånger du har läst denna bok. |
| Senast läst | Datumet du senast läste klart denna bok. *Valfri.* |
| Betyg | Ditt personliga betyg. *Valfri.* |
| Favorit | Om den här boken är markerad som favorit. |

## System- och källfält

| Fält | Beskrivning |
|------|-------------|
| Källa | Varifrån katalogposten kom (t.ex. Importerad, Manuell, ISBN-sökning). *Valfri.* |
| Medielänk | En URL till relaterade medier eller förlagets sida för denna bok. *Valfri.* |
| Kategorier | De samlingskategorier den här boken tillhör (t.ex. Skönlitteratur, Serier). Hanteras i filterpanelen. |
| Tillagd | Datum och tid då posten skapades i BookDB. Sätts automatiskt. |
| Uppdaterad | Datum och tid då posten senast ändrades. Uppdateras automatiskt vid sparande. |
