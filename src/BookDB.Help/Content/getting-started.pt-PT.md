# Primeiros passos

Bem-vindo ao BookDB — um catálogo pessoal para a sua coleção de livros. Este guia mostra as primeiras coisas que vai querer fazer.

## Adicione o seu primeiro livro

A forma mais rápida de adicionar um livro é pelo ISBN:

1. Prima **Ctrl+N**, ou utilize **Ficheiro → Novo Livro**, ou o botão de adicionar na barra de ferramentas.
2. Escreva ou digitalize o **ISBN** e prima **Enter**.
3. O BookDB procura o livro em várias fontes públicas e abre a caixa de diálogo **Combinar e rever**, onde escolhe o título, os autores, a capa e outros dados a manter.
4. Escolha uma **coleção** para o livro e clique em **Guardar** — ou em **Guardar e abrir o editor** para preencher mais.

Não tem o ISBN? Escolha **Introduzir os dados manualmente** na mesma caixa de diálogo e escreva o título, os autores e os restantes campos.

## Importe uma coleção existente

Já tem uma biblioteca no Readerware? Não precisa de a introduzir novamente:

- Utilize **Ficheiro → Importar cópia de segurança do Readerware…**, ou o botão **Importar cópia de segurança do Readerware** no ecrã de biblioteca vazia.
- Indique ao assistente um **.zip de cópia de segurança** do Readerware ou a **pasta de cópia de segurança** extraída.
- O assistente mostra uma pré-visualização do que encontrou, deixa-o escolher uma coleção de destino e depois importa.

Consulte o separador **Guia de importação** para um passo a passo.

## Ligar-se a uma base de dados

Por predefinição, o BookDB mantém a sua biblioteca num ficheiro local neste computador — não há mais nada a configurar.

Para partilhar uma biblioteca entre dispositivos, ou para utilizar PostgreSQL / MySQL, abra **Definições → Base de dados** (ou o botão **Ligar a uma base de dados** no ecrã de biblioteca vazia) e introduza os dados de ligação. O separador **Bases de dados remotas** explica as opções.

## Organize a sua biblioteca

- As **coleções** agrupam os seus livros (Ficção, Não ficção, Banda desenhada …). Defina uma ao adicionar um livro e altere quando quiser, editando o livro.
- **Edite** um livro para ajustar os seus dados, a capa e os colaboradores. Os nomes de autores e outros colaboradores reutilizam pessoas existentes à medida que escreve, para que a mesma pessoa não seja duplicada.
- **Pesquise e filtre** a partir da barra de ferramentas para encontrar tudo à medida que a sua biblioteca cresce.

## Para onde ir a seguir

- **Glossário de campos** — o que significa cada campo.
- **Guia de importação** — a importação em detalhe.
- **Sobre as fontes de dados** — de onde vêm os dados dos livros.
- **Bases de dados remotas** — partilhar uma biblioteca entre dispositivos.
